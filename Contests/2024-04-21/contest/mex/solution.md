题目给出一个$mex(S)$=序列$S$中未出现的第一个非负整数，其中$S$为$0$到$n-1$的非负整数序列。
要求由$0$到$n-1$组成的所有序列中，其序列的所有连续子串的$mex$值和，最大和最小。

首先对于最小值，可以分析长度为1的子串，除开$0$以外都是$mex$值都是0，所以长度为1的子串的值和固定位$1$；
对于长度为n，值和固定位$n+1$

对于其他长度的串，考虑要让尽可能多的子串$mex$小因此要构造这种情况：
只要是$0$没有出现，其$mex$值则必为最小的$0$，因此要值最小不包含0的子串要最多

所以长度为2的串，可以让$0$只在一个子串中出现这样其他子串$mex$都是$0$，剩下一个唯一包含$0$的子串$mex=1$。
例如：$1 4 3 2 0$
长度3和以上的串类似，都是只要让$0$只出现在一个子串中即可让其他子串$mex=0$
因此一定可以对任意n长度构造一个序列：在任何$2$到$n-1$长度的连续子串中，只有一个包含$0$，这样值必定是最小的。
因此最小值为 $ 1 + n-2 + n $

对于求最大值的情况，由于$mex$的性质要构造某个长度下的子串$mex$值尽量大，就需要$0$开始连续的整数都出现。
对于n=5的情况可以考虑如下构造：
长度$l$：最多只有一个子串可以包含完整的$0 ... l$，其他最多包含连续的$0 ... l-1$、$0 ... l-2$等等，因此最大值情况子串的$mex$值可以为$l+1, l, l-1, ... 1$

所以有：
长度2：${0, 1}$，只有包含$0, 1$和$0$的子串有值为$2$和$1$
长度3：${0, 1, 2}$，只有包含$0, 1, 2$和$0, 1$、$0$的子串有值$3$、$2$、$1$且为最大情况
长度4：${0, 1, 2, 3}$，由于长度限制，最大值为包含$0,1,2,3$和$0,1,2$的子串使得和最大，为$4$和$3$
其中一个可以对应的序列就是$4 3 0 1 2$

因此不难总结出最大情况的规律：
子串长度$l$的mex最大值: $l + l-1 + l-2 ... + l-j$，$j=min(i, n-i+1)$
$l = 1, 2, 3, ..., n$
最大值将以上数值累加即可.
