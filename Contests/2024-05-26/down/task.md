## 第一题 花园 garden.cpp
时间限制：1.0秒， 内存限制 256MB

最近雨水严重，J先生的花园积了很多水。J先生的花园可以看做一个$n \times m$的表格，每个格子都有一个非负整数的高度, 格子内部是平整的，且不会漏水。相邻格子的高度可能不同。 花园以外可以看做高度为0的连通区域，并且有下水管道。花园完全露天。昨晚下了一夜的雨，可以认为降雨量已经超过了花园中任何一个格子的高度。请问花园里现在有多少积水。积水量可以看做是每个格子储水的高度之和。


## 输入格式
第一行两个整数，$N,M$,分别表示花园的长和宽。
接下来有$N$行，每行有$M$个整数，表示花园中每个格子的高度。 

## 输出格式
一个整数，表示花园中的储水量。

## 第二题 外星人砖堆 brick.cpp
时间限制：1.0秒  内存限制：256MB

外星人麦圈好久都没有出现了。但是外星人砖块堆似乎来了。工地上某个区域围了一些砖块，这些砖块整齐排列，刚好有$n$堆，编号为$1, 2, \dots, n$,它们构成一个环，即$\forall i \lt n$, 第$i$堆与$i+1$堆相邻，第$n$堆与$1$堆相邻。第$i$堆砖块有$h_i$块砖。所有的砖块形状、大小都完全相同，且都是统一的方式平铺的。
每天早上醒来，砖块堆的高度都在发生变化，没有人知道是谁在操作这些砖块。好奇的你决定彻夜蹲守观察，凌晨三点，你发现了一个外星小孩，它突然出现在砖块堆围成的环中，它似乎可以用意念来操作这些砖块，每次它都能够让相邻的两个砖块堆同时增加或同时减少一个单位。它似乎可以无限的使用意念去操作。虽然你在旁边吓得大气不敢出，但外星人小孩还是发现了你，你似乎感觉到它在跟你说话，虽然你没有听到任何声音。但是你的大脑清晰的捕捉到了它发过来的讯息：”我每次只能改变相邻两堆砖块的高度，同时增加一个单位或者减少一个单位。我想要用最少的次数，将这些砖块全部变成同样的高度。如果有多种高度可以选择，在保证操作次数最小的前提下，高度越小越好。但在任意时刻，任意堆的砖块高度都不能为负。地球人，你能帮我吗？“
为了不给地球人丢脸，你爽快的答应了。作为一个OIER，你掏出了笔记本，写起了代码。

## 输入格式

第一行一个整数$N$.
接下来有$N$堆砖块，给出每堆砖块的高度。 

## 输出格式

两个整数，第一个整数表示最终砖块的高度，第二个整数表示最小要操作的次数。 如果不能达到同样的高度，则输出两个-1. 
如果有多种方案，能够用最小的操作次数达到同样的高度，请输出最小的高度。

## 输入样例1
```plain
4
3 2 8 9
```

## 输出样例1

```plain
2 7
```


## 输入样例2
```plain
3
1 2 3
```

## 输出样例2

```plain
2 2
```

## 第三题 演奏 （perform.cpp)
时间限制：1.0秒  内存限制： 64MB

## 题目描述 
麦加在弹钢琴，但他的演奏却和一般人有点不一样，完全是乱弹。钢琴上一共有N个琴键，编号为1～N。对于每个琴键
，被他按下的概率是相等的，且和之前按下的琴键无关，是一个独立随机事件。现在指定一段包含M个音符的音乐，如
果想要听到这M个音符，麦加按键的期望次数是多少次？不仅如此，你还需要求出这个M个音符的每一个前缀第一次出现
时的期望按键次数。由于结果可能很大，你需要模$10^9+7$。 


## 输入格式
第一行输入一个整数$N$，表示钢琴上的按键数。

第二行输入一个整数$M$,表示指定的一段曲子包含的音符个数。

第三行有$M$个整数，第$i$个整数为$A_i(1 \leq A_i \leq N)$, 表示曲子的第$i$个音符。

## 输出格式 
输出$M$行，每行一个正整数，第$i$表示第一次出现曲子的前$i$个音节的期望按键次数，答案模$10^9+7$.

```

```

## 样例输入 #3

## 第四题 蚂蚁 （ants.cpp)
时间限制：4.0秒  内存限制：512MB

## 题目描述

在一个二维平面上，有 $N$ 个点，第 $i$ 个点坐标为 $(X_i,Y_i)$，每个点都有一只蚂蚁，第 $i$个点处的蚂蚁编号为$i$。从第$0$时刻开始，每只蚂蚁会在平面上移动，他们可以选择东南西北：

- 如果选择东，则 $t$ 时刻过后坐标位置从 $(x,y)$ 变为 $(x+t,y)$。
- 如果选择西，则 $t$ 时刻过后坐标位置从 $(x,y)$ 变为 $(x-t,y)$。
- 如果选择南，则 $t$ 时刻过后坐标位置从 $(x,y)$ 变为 $(x,y-t)$。
- 如果选择北，则 $t$ 时刻过后坐标位置从 $(x,y)$ 变为 $(x,y+t)$。

$t$ 不一定是整数。 

在$0$时刻，只有$1$号蚂蚁接收到了讯号。

如果某一个时刻，蚂蚁$a$和蚂蚁$b$在同一个坐标上，且$a$得知了讯息，则$b$蚂蚁也将得到讯息。

请问第$10^{100}$个时刻，最多会有多少只蚂蚁接收到信息。



## 输入格式

第一行一个整数 $N$ 代表点的数量。

接下来 $N$ 行每行两个整数 $X_i,Y_i$ 代表一个点的坐标。

## 输出格式

一行一个整数代表答案。

## 样例 #1

### 样例输入 #1

```
4 3
```

### 样例输出 #1

#### 数据规模与约定

**本题采用捆绑测试。**

- Subtask 1（5 pts）：$N \le 7$，满足性质 A。
- Subtask 2（8 pts）：$N \le 15$，满足性质 A。
- Subtask 3（6 pts）：$N \le 100$，满足性质 A 和 B。
- Subtask 4（6 pts）：$N \le 100$，满足性质 A。
- Subtask 5（12 pts）：$N \le 3000$。
- Subtask 6（32 pts）：满足性质 A。
- Subtask 7（31 pts）：无特殊限制。

对于 $100\%$ 的数据，$1 \le N \le 10^5$，$0 \le X_i,Y_i \le 5 \times 10^8$，点的坐标互不重合。

其中性质分别为：

- 性质 A：$X_i \ne X_j$，$Y_i \ne Y_j$。
- 性质 B：$1$ 号点坐标为原点。
