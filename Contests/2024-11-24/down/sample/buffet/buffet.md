# buffet

## 题目背景

$CZN$在2024年高中数学联赛中取得佳绩，所以他要请大家吃自助餐。

## 题目描述

为了不踩坑，$CZN$决定先去试吃一次。

餐厅一共有$n$道菜，$CZN$对于每一道菜都有一个不满度，
因为桌子太小，所以厨师会分批次上菜，$CZN$对于每一批菜的不满度是这批菜中最大的不满度与最小的不满度的差值，如果不满度总和小于等于$k$，$czn$就会在这宴请他的朋友。

作为厨师的你，已经提前知道了$CZN$对于每道菜的不满度，问有多少种上菜的方案使得$CZN$的总不满度小于等于$k$。

由于上菜的方案可能太多了，你需要输出方案数模$998244353$的值。

## 输入格式

第一行两个整数$n$,$k$。

第二行$n$个整数表示$CZN$对于每道菜的不满度。

## 输出格式

一个整数表示取模后的方案数。

## 样例 #1

### 样例输入 #1

```
3 2
1 4 6
```

### 样例输出 #1

```
2
```

## 提示

对于 $20\%$ 的数据，满足 $1\leq n\leq 20$，$1\leq k\leq 500$，$1\leq a_i\leq 300$

对于 $40\%$ 的数据，满足 $1\leq n\leq 100$，$1\leq k\leq 500$，$1\leq a_i\leq 300$

对于所有数据，满足 $1\leq n\leq 200$，$1\leq k\leq 1000$，$1\leq a_i\leq 500$
