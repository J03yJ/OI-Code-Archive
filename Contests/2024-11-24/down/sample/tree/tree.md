## 题目描述

给定一个$n$个点，$m$条边的图，点初始全为白色。每次可以选择两个相同颜色的点，将它们的颜色变为相反的颜色。问最少进行几次操作可以将所有点变为黑色；若无解输出$-1$。

## 输入格式

两个整数$n$,$m$。

$m$行，每行两个整数代表边的两个端点

## 输出格式

一个整数表示答案

## 样例 #1

### 样例输入 #1

```
6 5
1 4
3 4
1 6
2 1
1 5
```

### 样例输出 #1

```
-1
```

## 样例 #2

### 样例输入 #2

```
6 6
5 1
6 4
3 5
5 6
1 3
4 2
```

### 样例输出 #2

```
3
```

## 提示

$1\leq n \leq 5 \plus 10^5$

保证对于$30\%$的数据$m=n-1$。

另外$70\%$的数据$m=n$。
